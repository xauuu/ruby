## Cài đặt  
``` bash
git clone https://github.com/xauuu/ruby.git
```
``` bash
bundle install
```
``` bash
yarn add bootstrap jquery popper.js
yarn add @fortawesome/fontawesome-free
```
``` bash
sudo service mysql start
```
``` bash
rake db:create
rake db:migrate
```
``` bash
mysql -u [username] -p xau_development < ruby.sql
```
``` bash
rails s
```
